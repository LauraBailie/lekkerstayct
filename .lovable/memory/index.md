# Project Memory

## Core
LekkerStay CT - Cape Town rental transparency platform. Ocean blues, mountain greens, SA gold accents.
Space Grotesk headings, Inter body. Cape Town slang throughout (howzit, lekker, awê, eish, sharp-sharp).
Lovable Cloud for auth (email + Google) and database. Real-time subscriptions on rentals and pulse_reports.
Built by Laura Bailie at SA Startup Week.
No "kak" ever. Use "bru" sparingly. Tone: warm, professional, community-focused.
